# Assets

Place the following files here:

- `hero-banner.png` — README hero image (recommended: 1600x400, dark background with framework name and key tech logos)
- `allure-screenshot.png` — Screenshot of the Allure report dashboard
- `pipeline-screenshot.png` — Screenshot of a green GitHub Actions run
- `demo.gif` — GIF showing tests running in headed mode (record with ScreenToGif or LICEcap)

## How to create the hero banner

Use [Canva](https://www.canva.com) or [Figma](https://www.figma.com) with:

- Dark gradient background (#1a1a2e to #16213e)
- Framework name in bold white
- Tech logos: Playwright, TypeScript, Cucumber, Docker, GitHub Actions
- Dimensions: 1600 x 400px
- Export as PNG
